Strings and Chars

Writing text and their commands

Strings and characters are datatypes which allow the programmer to utilize words or letters and assign them to variables. By using the String datatype, users can type in words or sentences, which through many different commands can be altered and read by the program. Declaring a String datatype is no different from declaring previous datatypes such as integers. The only difference there is is what is assigned to the variable. The following code outlines the difference.

public class Strings {
public static void main(String args[]) {

int num = 1;
String sentence = "Hello, World. This is a String";
}
}

As you can see, when Strings are declared you assign letters or words to them which are in quotation marks. Below I will list several commands that can be used with Strings or chars. Run the code and change the variables in the print statement to see what each really does.

public class Commands {
public static void main(String args[]) {

String name, nameSub, nameSub2, nameLower, nameUpper, nameTrim, nameRep, nameRepAll;
boolean nameCompare, nameCompareIgnore;
char charAt;

name = "Michael Smith";
// regular string

nameSub = name.substring(3);
// removes the first 3 characters

nameSub2 = name.substring(5,7);

// removes all characters except for the ones in the range provided (5th character to 7th character)

nameLower = name.toLowerCase();
// converts string to lowercase

nameUpper = name.toUpperCase();
// converts string to UPPERCASE

nameTrim = name.trim();
// trims the spaces in front and behind the string

nameRep = name.replaceFirst("i", "z");
//replaces the first occurrence of the string provided with the second string provided

nameRepAll = name.replaceAll("i", "z");
//replaces all occurrences of the first string provided with the second

nameCompare = name.equals("michael smith");
// compares two strings, returns a boolean value

nameCompareIgnore = name.equalsIgnoreCase("michael smith");
// compares two strings and ignores CAPITALIZATION, returns a boolean value

charAt = name.charAt(0);
// returns the letter at the specified index

System.out.println(name);
}
}
