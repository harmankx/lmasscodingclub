Loops

The While Loop

What are Loops? Well loops iterate a sequence of commands; they can loop for a certain amount of time, as long is something is true, or forever. There are 3 different types of loops: while loops, do while loops and for loops.

The first type of loop is a while loop. While loops loop code as long as a certain condition is true. For example, as long as a variable is less than a number. A while loop could be written like so:

public class While_Loop {
public static void main(String args[]) {

int num = 1;

while (num <= 10) {
System.out.println("This is a looping statement");
num++;
}
}
}

This code will loop “This is a looping statement” and add 1 to num as long as num is less than or equal to 10.

The Do-While Loop

A Do while loop first executes the code once and then checks if the statement is true, if so it will continue to loop otherwise it would end there. It is written like this:

public class Do_While {
public static void main(String args[]) {

int num = 1;

do {
System.out.println("This is a looping statement");
num++;
} while (num <= 10);
}
}

This code will output the exact same thing as the while loop, but the only difference is that do loops will always run the code in the loop at least once, and then decide whether to loop it or not, unlike while loops which will only execute the code if the condition is first met.

The For Loop

A for loop executes code for a certain amount of iterations which you can specify. It uses a initialized variable as its starting point, then the condition, and finally the increment the variable increases/decreases at. For loops are written in the format (initilaize variable; stopping point; increment). For example, the for loop that performs the same task as the previous loops looks something like this:

public class For_Loop {
public static void main(String args[]) {

for (int i = 0; i <= 10; i++) {
System.out.println("This is a looping statement");
}
}
}
