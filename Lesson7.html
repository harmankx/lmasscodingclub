Methods

What are Methods? A method is a collection of statements that can be executed when called by name. Another way to think of a method is a mini program separated from the main program until called inside the main method (Note: Methods should always be written outside of the "main" method). There are two types of methods: one that returns a value and one that does not. A method that returns a value must have a data type, for example ”int”, “double”, etc. Let’s say you want to create an integer method named “fivePlusSeven” that added 5 and 7 and returned the value, you would write that like this:

static int fivePlusSeven() {
int sum = 5 + 7;

return sum;
}

In this case, return means it will produce the value of “sum”, a.k.a 12. Calling it in the main method looks something like this :

public static void main(String args[]) {
System.out.println(fivePlusSeven()); //prints the value 12
}

The other method, the one that does not return a value, does not need to be declared with a data type. Instead, it must be declared with the keyword “void”, as it will not produce any value at the end. It is recommended that you do not use this version when evaluating numbers. An example for using this type of method would be printing a series of strings. Let's say you want to make a void method named “iLoveCS” that prints three strings on different lines, you would write it like this:

static void iLoveCS() {
System.out.println("I");
System.out.println("Love");
System.out.println("Computer Science");
}

Notice there is not a “return” command, that’s because there is no value to return but rather there are commands to execute. Calling it in the main method would look like this:

public static void main(String args[]) {
iLoveCS();
}

Modifiers

There are a few modifiers you need to know about. Modifiers are essentially levels of accessibility for variables and methods. Some modifiers that are useful include: public, static, and void. Public means the method can be accessed through classes. Static means the method is not associated with any particular instance of the program but rather can be accessed at any point. Void means that the method produces no end value.

You can also set parameters in your method. Parameters are values/variables that are being passed into the method. They are written in the parentheses of the method. For example, if you want to make a method that adds two numbers that you’ve assigned in your main method, it would look something like this:

public class Methods {
public static void main(String args[]) {

int firstNum = 5;
int secondNum = 16;
System.out.println(addTwoNumbers(firstNum,secondNum));
//firstNum and secondNum are passed into the method
}

public static int addTwoNumbers(int num1, int num2) {
int sum = num1 + num2;
//equivalent of firstNum + secondNum
return sum;
}
}
