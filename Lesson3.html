Variables and User Input

Variables

A variable is a storage location, where you are able to store values into. Variables have data types, meaning they have classifications which define those variables, for examples one variable may only store integers, while another may only store decimals or Strings. The different primitive, or simple, data types include:

int: This is the type of variable that only stores integers. Ex. 1 or 4 or 245843

double: This is the type of variable that stores decimals Ex. 3.1415 or 43.654764

boolean: This data type stores whether something is either true or false Ex. boolean cool = true;

char: This stores only singular letters Ex. a, or e, or i, or g

There is another data type known as a String that stores sentences, however this is a complex data type and I will go more in depth with this concept in future lessons.

To declare a variable, you must use the following format:

datatype variableName = variableValue;

For example:

public class Variables {
public static void main(String args[]) {

int x = 15;
double pi = 3.14159265;
boolean on = true;
char letter = 't';

}
}

Printing Variables

To print out a variable, we go back to the command System.out.println(), but this time there is one difference. To output a variable to the user you MUST NOT put in parentheses, however you should put it in the brackets. For example:

public class Variables {
public static void main(String args[]) {

int x = 15;
double pi = 3.14159265;
boolean on = true;
char letter = 't';

System.out.println(x);
System.out.println(pi);
System.out.println(on);
System.out.println(letter);

}
}

These print statements will print out whatever is in the variables, isn't that neat!

Concatenation

Concatenation is how you can combine strings and/or variables together which can prove to be useful in print statements!. To do this we can use the “+” operator within the brackets of a print command in combination with another string or variable. For example:

public class Concat {
public static void main(String args[]) {

int apples = 6;
System.out.println("There are " + apples + " apples!");

}
}

User Input

We have talked about outputs and how to output things to the user. Now we will talk about how to receive input from the user and how we can use this information in things such as assigning values to a variable. First we must import the Scanner library:

import java.util.Scanner;
public class UserInput {
public static void main(String args[]) {

}
}

Next we are going to declare a Scanner, which is what we use to receive input from the user:

import java.util.Scanner;
public class UserInput {
public static void main(String args[]) {

Scanner input = new Scanner(System.in);

}
}

Note how this declaration of a Scanner is written. We can refer back to variables to understand this more. In a sense, this follows the "datatype variableName = variableValue;" format we looked at before, where Scanner is the datatype, input is the name of the variable, and "new Scanner(System.in);" is the value of the variable.

It is "new Scanner(System.in)" because we are declaring a brand new scanner variable that can take in user input. If we look at "System.in", we see that it looks very simmilar to "System.out.println()", this is because System.out is an output stream that outputs to the program, while System.in is an input stream which brings data into the program from the user.

To set a variable to accept user input, we must utilize the Scanner variable "input" we have created. There are different ways to recieve user input depending on the datatype for the variable. This is a list of the commands:

intVarible.nextInt() - if you want the user to input an integer

doubleVariable.nextDouble() - if you want the user to input a double

booleanvariable.nextBoolean - if you want the user input a boolean value

Let’s say you want to make a program that asks for the amount of apples the user has. It would be written as follows:

import java.util.Scanner;
public class Apples {
public static void main(String args[]) {

Scanner input = new Scanner(System.in);
int apples;

System.out.print("How many apples do you have? ");
apples = input.nextInt();
System.out.print("You have " + apples + " apples!");
}
}

When executed, this program will first ask how many apples the user has, and then the user will be able to input a value. This value will be then assigned to the varible "apples" through the ".nextInt()" function. We then print to the user that they have the amount of apples that they entered by using concatenation.
